📥 Importer des Transactions CSV

📋 Format requis

Votre fichier CSV doit contenir :

  • Date : Format DD/MM/YYYY ou YYYY-MM-DD
  • Description : Texte décrivant la transaction
  • Montant : Nombre (positif = revenu, négatif = dépense)
  • Catégorie : (Optionnel) Nom du budget à affecter

Exemple :
Date,Description,Montant,Catégorie
15/01/2025,Salaire,2500,
16/01/2025,Courses,-85.50,Alimentation
